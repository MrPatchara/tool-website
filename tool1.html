PYTHON TO EXE CONVERTER

โปรแกรมแปลงไฟล์ Code python.py ไปเป็น .exe สําหรับ run บน windows.



สิ่งที่ต้องมีก่อนใช้งานได้!

-Python3

-Pyinstaller หากยังไม่มี ให้ติดตั้งก่อนโดย cmd คําสั่ง > pip install pyinstaller


วิธีใช้งานโปรแกรม


-รันโปรแกรม Program_py_to_exe.bat


-เลือกไฟล์ .py ที่ต้องการแปลง


-เลือกไดเรกทอรีที่ต้องการเก็บไฟล์ .exe


-(ถ้ามี) เลือกไฟล์ไอคอน .ico


-เลือกตัวเลือกการสร้างไฟล์เดียว (--onefile) หรือการปิดหน้าต่างคอนโซล (--windowed)


-กดปุ่ม Convert to EXE


-กดปุ่ม Contact Developer เพื่อดูข้อมูลติดต่อผู้พัฒนา


-ใช้ปุ่ม Reset เพื่อรีเซ็ตฟอร์ม (โปรแกรมจะบันทึกการตั้งค่าเมื่อปิดโปรแกรม)
